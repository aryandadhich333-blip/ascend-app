# ASCEND - AI-Powered Productivity RPG

## Overview
ASCEND is a premium self-improvement platform that transforms productivity into an RPG adventure. Users level up by completing real-life tasks, earn coins, unlock rewards, and track progress across multiple life improvement areas.

## 🎮 Features

### Core Gamification
- Level-based progression system
- XP rewards for task completion
- Coin economy
- Streak tracking
- Achievement system
- Customizable avatars and themes

### 🎯 Task & Quest System
- AI-generated personalized quests
- Multiple difficulty levels
- Task categories
- Detailed instructions and tips
- Success criteria and reflection questions

### 🏆 Tracker (Analytics)
- Daily, weekly, monthly XP tracking
- Skill tree progress
- Achievement tracking
- Statistics dashboard
- Weekly AI analysis reports

### 🤖 Coach (AI Assistant)
- Daily routine planning
- Personalized quest generation
- Self-improvement guidance
- Schedule optimization

### 🎁 Reward System
- Freedom time unlocking
- Hobby/reward creation
- Shop with collectible pets
- Theme and avatar unlocks

## 🛠️ Tech Stack

### Frontend
- React.js 18+
- TypeScript
- Tailwind CSS
- Framer Motion
- Chart.js

### Backend
- Node.js + Express
- MongoDB
- JWT Authentication
- OpenAI API
- Socket.io

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/aryandadhich333-blip/ascend-app.git
cd ascend-app

# Install all dependencies
npm install

# Setup environment
cp .env.example .env

# Start development
npm run dev
```
